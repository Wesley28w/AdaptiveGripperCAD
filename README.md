# AdaptiveGripperCAD
My First CAD Project where I built something from scratch with no guide.
It's an adaptive gripper that uses a servo motor as the driver, rotating a 24T Mod 0.5 spur gear into two other 36 T Mod 0.5 spur gears. Using a second upper joint and revolutes on each of the two connection points between said joints and the gripper, we create an adaptive gripper mechanism that closes inward and moves forward in one clean motion.
